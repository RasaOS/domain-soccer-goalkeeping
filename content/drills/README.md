# Drill Library

The practical companion to the [syllabus](../syllabus/README.md). The full
library lives in [`library.md`](library.md) — every named drill as a card:

> **### Drill Name** → **Objective** · **Setup** · **Execution / progression** · **Coaching cues** · **Syllabus area**

Cards are grouped by syllabus area, so a coach working a category can jump
straight to its drills. Each card cross-links back to its
[syllabus](../syllabus/README.md) reference and the
[methodology](../framework/training-methodology.md) it's run under.

Run drills inside one of the two delivery methods —
**Simple-to-Complex** or **Whole-Part-Whole** — and always finish at a
match-realistic rep (see
[training-methodology](../framework/training-methodology.md)).

## Sources

Named drills come primarily from the Goalkeeper Plan 2015–2016 (Stalk,
Steal Ground, Power Step, the dive family, the catch family, set-piece
management); technique-implied exercises come from Robo ("The GK Trainer").
See [`../reference/`](../reference/README.md).

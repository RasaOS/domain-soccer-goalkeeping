# Technical Syllabus

The **what** of goalkeeping — the technical and tactical curriculum, in
nine categories. Each file is a self-contained reference: why it matters,
the technical progression (foundational → advanced), coaching points &
cues, common faults & corrections, emphasis by development stage, related
[drills](../drills/library.md), and key [terms](../vocabulary.md).

| Category | File | Essence |
|---|---|---|
| Handling / Catching | [`handling-catching.md`](handling-catching.md) | Hands first; ball at the body vs away; concave hands behind the ball; lock it into the chest. |
| Footwork / Movement | [`footwork-movement.md`](footwork-movement.md) | The set position; first step off the foot closest to the ball; shuffle vs sprint; getting "set" before the shot. |
| Shot-Stopping / Diving | [`shot-stopping-diving.md`](shot-stopping-diving.md) | Collapse, step, and extension dives; near-body vs extended; catch first, parry as a last resort. |
| Aerial / Crosses | [`aerial-crosses.md`](aerial-crosses.md) | High balls and flank service; "Can I win the ball?"; catch vs punch vs parry; attacking the highest point in traffic. |
| 1v1 / Breakaway | [`one-v-one-breakaway.md`](one-v-one-breakaway.md) | The decision to engage; timing; body shape / making yourself big; when to hold. |
| Distribution | [`distribution.md`](distribution.md) | Throws, punt, dropkick, goal kicks; fast break vs keep possession; "when in doubt, kick it out." |
| Positioning / Angles | [`positioning-angles.md`](positioning-angles.md) | Narrowing the angle; depth off the line; near-post priority; sweeping; hold position and **react**, don't anticipate. |
| Playing With Feet | [`playing-with-feet.md`](playing-with-feet.md) | Receiving back-passes, first touch under pressure, support angles, switching play — the modern sweeper-keeper. |
| Psychology / Mentality | [`psychology-mentality.md`](psychology-mentality.md) | Bravery, concentration, resilience to mistakes, command presence, the decision-making mindset. |

## Note

Categories are reference, not a strict order. Sequence them per the keeper's
[development stage](../framework/development-framework.md) — technique-first
for younger keepers, broadening to positioning and game-reading as they
mature.
